Convert Trello export to MindMap file for Coggle.

How to use: On Trello board, go to settings/export. Pick JSON. You can either save the result as a file or copy the text to clipboard. Either way, give what you want here and press convert. You'll receive a file to download. Save it and fire up Coggle. Create a new diagram and then drag and drop the saved file to it. That's it.
